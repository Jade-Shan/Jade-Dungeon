上传图片资源

因为本服务器是最便宜的服务器，没有空间可以上传图片。 图片只能用第三方图床来外链。

以前新浪微博的图片可以外链，现在已经不能用了。 可以找一些速度慢免费的第三方图床，比如：路过图床https://imgtu.com/

创建沙盘

注册登录

首先注册登录，过程略。

新建场景沙盘

每个沙盘的场景都以：「模组」、「地点」、「场景」作为唯一标识。比如：

「深水城大冒险」「海拉斯特的地下城」「山下一层」

第一个使用新的模组名称的用户将作为该模组的拥有者与维护者， 其他人无法编辑任何使用该模组名的场景沙盘。

编辑沙盘

沙盘上所有的元素按使用过程中操作的频率分类为：

  • 玩家
  • 非完家生物
  • 场景物品
  • 门
  • 墙壁
  • 图片资源

图片资源

图片资源只在创建沙盘时用到，使用的频率较低。

背景地图

点击图片资源的按钮，其中必须有一张名为map的图片，该图片是整个场景的地图背景：

编辑器会为新创建的场景生成一张默认的map图片。 点击该图片，下方的编辑区中可以把图片的URL换成你场景中需要用到的地图。

导入新图片资源

按下方的「创建新图像」按钮，会创建一个新的图片资源。 下方的编辑区中可以把图片的URL换成你场景中需要用到的图片。

墙壁元素

墙壁元素只在创建沙盘时用到，使用的频率较低。

墙壁用来遮挡玩家的视线，有线条、圆形、方块三种按钮用来建立三种不同的墙壁。

点击创建线条按钮后，再用鼠标点击沙盘， 拖动后放开会在该位置创建一个遮挡玩家的视线墙壁。

棋子元素

棋子元素在跑团过程会经常移动位置。 以下这些都是棋子元素：

  • 玩家
  • 非完家生物
  • 场景物品
  • 门

地图的编辑者可以通过在沙盘上点击或是在列表中选择的方式来选中棋子。 正文的编辑区域会显示棋子的属性，如位置、大小、图标、是否可见， 是否遮挡其他玩家的视线等。

创建棋子

在列表中选择对应的棋子列表，列表下方会显示创建不同类型棋子的按钮。

点击创建按钮后，再用鼠标点击沙盘会在对应的位置创建棋子。

控制棋子

  • 通过编辑位置属性或是按Ctril加鼠标拖动的方式可以移动棋子的位置。
  • 通过编辑大小属性或是按Alt加鼠标拖动的方式可以改变棋子的大小。
